# Kapi Event Organizer Information System

## Deskripsi Proyek
Sistem informasi terpusat berbasis web untuk Kapi Event Organizer. Sistem ini bertujuan mendigitalisasi proses kerja, mempermudah pengelolaan data klien/vendor, dan menyediakan otomatisasi budgeting acara.

* **Backend:** Embedded.js
* **Database:** MySQL

## Petunjuk Pengerjaan (SOP)
1.  **Branching:** Jangan coding di branch `main`. Buat branch baru untuk setiap fitur (contoh: `feature/login-page`, `fix/bug-budgeting`).
2.  **Commit Message:** Gunakan format jelas, misal: `[FE] Menambahkan form input klien`.
3.  **Pull Request:** Setiap fitur yang selesai harus di-merge melalui Pull Request (PR) dan di-review oleh anggota lain.
4.  **Kanban:** Pastikan status task di Trello/GitHub Projects selalu di-update (To Do -> In Progress -> Done).
